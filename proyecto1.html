Proyecto Fotomontaje

Recreación de escenas

En este proyecto recreamos las escenas de películas y series. Con el objetivo de recrearlas a partir de materiales reciclados o que cualquier persona tenga en casa.




Para recrear las escenas primero recolectamos los materiales que sirvieran para crear los elementos de cada imagen, posteriormente se tomaron varias fotografias y finalmente se procedió a la edicion fotográfica y montaje.
